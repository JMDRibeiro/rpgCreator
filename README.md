RPG Creator 
Sistema desenvolvido para a disciplina de LPW, lecionada pelo Prof. Leonardo Soares.
Discente : João Marcelo Duarte Ribeiro Sobrinho
Instituto Federal de Pernambuco campus Garanhuns.

Descrição:
Espera-se que, após concluido, este sistema possa ser utilizado para criação de Listas de Questões para
disciplina de Algoritmos, possibilitando que seja implementado um ambiente minimente gamificado, estimulando os 
estudantes a praticarem os exercicios e, portante, compreenderem melhor os conteúdos.
